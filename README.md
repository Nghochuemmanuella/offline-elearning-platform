# Offline E-Learning System with Local AI
A final year project focused on providing educational content in areas with limited internet access.

## Features (Planned)
- **Offline First**: Works without internet using PWA technology.
- **Local Database**: Student progress saves locally via PouchDB.
- **Local AI**: Integrated Ollama for offline tutoring (Phase 2).

## Tech Stack
- React + Vite
- PouchDB
- Service Workers (Vite PWA Plugin)
